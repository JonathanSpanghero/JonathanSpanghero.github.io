Meus Projetos

Sensor d'água para sistemas hidroponicos

Protótipo em pequena escala de um sistema hidropônico automatizado, projetado para facilitar o cultivo de plantas sem solo, identificando o nível de água e controlando automaticamente o seu fluxo e volume.

Desligamento remoto veicular

Aplicativo mobile em Kotlin que permite o controle remoto de dispositivos utilizando comunicação GSM. Com ele, é possível controlar remotamente o funcionamento de veículos que tenham o módulo de comunicação instalado.

Tradução de LIBRAS com visão computacional

Aplicativo experimental que utiliza visão computacional para traduzir sinais em LIBRAS para texto, ajudando a criar uma comunicação mais inclusiva para deficientes auditivos.

Obi: Cardseekers
